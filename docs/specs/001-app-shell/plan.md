# Plan — 001 App Shell

This spec establishes the frontend application foundation.

Implementation should be performed in phases.

---

# Phase 1 — Project Initialization

Create Angular application structure and configure dependencies.

Deliverables

- Angular project initialized
- Tailwind configured
- Angular Material installed
- base folder structure created

Outcome

Application compiles successfully.

---

# Phase 2 — Application Layout

Create the global layout structure.

Deliverables

- AppShell component
- AppHeader component
- Page container layout

Outcome

Application displays header and page content area.

---

# Phase 3 — Routing

Configure Angular router.

Deliverables

- root route
- recipe route `/r/:id`
- placeholder pages

Outcome

Router navigation works.

---

# Phase 4 — Theme System

Implement theme switching.

Deliverables

- theme toggle component
- light/dark theme styles
- persistence in local storage

Outcome

User can toggle themes.

---

# Phase 5 — Responsive Layout

Ensure layout works across devices.

Deliverables

- mobile layout
- tablet layout
- desktop layout

Outcome

Application is responsive.

---

# Completion Criteria

Application shell loads successfully and provides foundation for next spec.
